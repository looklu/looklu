video、audio元素

audio元素与video元素用法相同,其中audio支持mp3格式文件,video支持mp4、webm格式

            布尔属性: 某些属性,只有两种状态:1.不写,2.取值为属性名;布尔属性在H5中可以不用书写属性.
            controls: 布尔属性,控制控件的显示.
            autoplay: 布尔属性,自动播放.
            muted: 布尔属性,静音播放.
            loop: 布尔属性,循环播放.

ol、ul和dl

  1. ol: ordered list 有序列表
  2. ul: unordered list 无序列表
  3. li: list item
定义列表
通常用于一些术语的定义
dl: definition list
dt: definition title
dd: definition description

容器元素

语义化元素便于搜索引擎的识别与后续开发,div元素无任何语义

元素的包含关系由元素的内容类别决定

            header: 通常用于表示页头,也可以用于表示文章的头部.
            footer: 通常用于表示页脚,也可以用于表示文章的尾部.
            article: 通常用于表示整篇文章.
            section: 通常用于表示文章的章节.
            aside: 通常用于表示侧边栏.
            nav: 通常用于表示导航栏.
