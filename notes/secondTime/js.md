## 运算符
```
运算符：
    1. 算术运算符 + - * / % ++ --
    2. 赋值运算符 = += -= *= /= %=
    3. 比较运算符 > >= < <= == === != !==
    4. 逻辑运算符 && || !
    5. 条件运算符 ?:
    6. 字符串运算符 +
    7. 位运算符 | & << >> ~ ^
```
### 运算符优先级

1. 括号()[]
2. 求反 负数(-x) 求反(!x)
3. 自加、自减 x++(x值加1,返回原来的x值) ++x(x值加1,返回后来的x值)
4. 乘除
5. 加减
6. 比较
7. 赋值
